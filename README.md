# E-commerce Sales Analytics SQL Project 🛒

## Overview
Designed and built a relational database for an e-commerce platform to track customers, products, orders, and sales. Used advanced SQL JOIN queries to generate business insights.

## Database Schema
5 Tables: `customers`, `products`, `orders`, `order_items`, `categories`

## Key SQL Queries
1. **Top Selling Products:** Identified best-performing products using JOIN + GROUP BY
2. **Customer Lifetime Value:** Calculated total spend per customer using 4-table JOIN
3. **Monthly Sales Report:** Generated month-wise order trends using DATE functions

## Skills Demonstrated
SQL, SQLite, Database Design, Data Analysis, JOINs, Aggregate Functions, Business Intelligence

## Tools Used
DB Browser for SQLite, GitHub
